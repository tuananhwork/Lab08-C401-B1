Đáp án mẫu
Hoạt động 3 — Bộ cleaning rules (≥5 rule)
Áp dụng trực tiếp lên bảng dirty trên slide hoạt động
Clean / normalize
1. Trim whitespace cột content và notes.
2. Parse date đa định dạng → YYYY-MM-DD UTC.
3. Unicode NFC + thay thế ký tự thay thế cho “”.
Reject cứng
4. Drop row nếu content rỗng sau trim.
5. Drop duplicate theo khóa (doc_id, source, normalized_content_hash) — giữ bản mới nhất theo date parse được.
Flag / quarantine
6. Nếu date NULL nhưng content có → flag review_missing_date, không embed cho đến khi SME điền.
7. Log mọi drop/flag vào quarantine.csv kèm run_id.
Giảng viên

7 rule để dư cho nhóm chỉ nộp 5; nhấn quarantine vs silent drop.

Data quality as code
6 dimensions + Great Expectations / expectation suite
Completenesscontent not null
Accuracypolicy date đúng
Consistencysame key, same format
Timelinessfreshness theo SLA
Validityschema + contract hợp lệ
Uniquenesskhông duplicate record
Expectation suite
batch.expect_column_values_to_not_be_null("content") batch.expect_column_values_to_be_unique("doc_id") batch.expect_column_values_to_match_regex( "effective_date", r"^\d{{4}}-\d{{2}}-\d{{2}}$" ) if expectations_fail: raise PipelineHalt("bad data before agent")
fail pipeline if expectations fail
